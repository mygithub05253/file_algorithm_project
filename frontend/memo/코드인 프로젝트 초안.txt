프로젝트명 : 문서 자동 분류 및 폴더 생성 알고리즘 제작

목적 :
교내 생활 및, 업무 생활에 사용되는 문서를 직접 분리해야 하는 것이 불편하고 비효율적이라 생각하여 이것을 소프트웨어로 개발할 필요가 있다 생각함.

개요 :
단순한 exe 프로그램 또는 웹 사이트를 통해 작동하도록 구성
사용자가 문서나 파일을 올리는 경우 이 문서나 파일에 대해 프로그램에 탑재된 알고리즘이 분석하고 파일을 분류
zip파일 또는 폴더 형태로 정리하여 미리보기를 사용자에게 보여주고 다운로드를 가능하도록 구현

프론트 작업순서 :
1. 표지화면
- 표지화면 + 로그인, 회원가입 버튼
- 단순한 표지화면으로 구현

2. 기능 구현 화면
- 교내 생활, 업무 생활, 개인 파일 관리 화면 등 목적별 파일 보관소 구분
- 파일 추가, 삭제, 이동, 수정, 다운로드 등의 기능 구현 필요
- 파일 추가 시에는 단순하게 파일을 담은 뒤 추가 버튼을 누르면 프로그램에 탑재된 알고리즘에 따라 파일을 분류한 뒤 해당 파일을 자동으로 관련 보관소에 추가
- 파일 다운로드 시에는 zip 파일 또는 폴더 형태로 정리해서 미리보기를 화면에 보여준 뒤 다운로드 클릭하면 다운되도록 구현
=> 구글 drive에서 다운로드하는 원리 참고

3. 계정 회원가입 화면
- 학교 이메일 가입 권장
- 구글, 네이버, 카카오톡으로 가입 기능 구현(가능한 경우)
- 이메일, 문자메시지 인증 기능 구현
- 회원가입 성공 시 로그인 화면으로 이동

4. 계정 로그인 화면
- 아이디 혹은 비밀번호 불일치 시 안내 메시지 출력
- 로그인 성공 시 기능이 구현되어 있는 화면으로 이동
