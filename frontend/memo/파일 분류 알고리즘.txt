1. 파일 분류 알고리즘 개요
- 파일이 없로드 될 경우 다음 과정을 수행
1). 파일명을 가져와 공백을 제거 => 즉, 파일명을 단일 문자열로 변환
2) 현재 존재하는 폴더 목록을 확인
3) 파일명이 특정 폴더명을 포함하는 지 검사
	- 포함될 경우 "전체 파일"과 해당 폴더에 자동 추가
	- 포함되지 않을 경우 "전체 파일"과 "기타 파일" 폴더에 자동 추가

2. 파일 분류 알고리즘 상세 설명
1) classifyFile() 함수
- 파일명을 기반으로 폴더 자동 매칭

2) uploadFiles() 함수
- 파일 업로드 시 자동으로 해당 폴더에 저장

3. 파일 분류 알고리즘 동작 흐름
1) 사용자가 파일을 업로드
2) 파일명에서 공백 제거 후 문자열로 변환
3) 현재 존재하는 폴더 리스트 조회
4) 파일명이 폴더명을 포함 여부 검사
- 포함하는 폴더가 있는 경우 해당 폴더와 "전체 파일"에 추가
- 포함하는 폴더가 없는 경우 "전체 파일"과 "기타 파일"에 추가
5) 파일을 해당 폴더에 저장 및 화면에 표시

4. 파일 분류 알고리즘 동작 예시(폴더 : 전체 파일, 기타 파일, 컴퓨터, 업무, 프로젝트)
- 컴퓨터 수리 보고서.pdf => "전체 파일", "컴퓨터"
- 업무 계획서.docx => "전체 파일", "업무"
- 데이터 분석 결과.xlsx => "전체 파일", "기타 파일"
- 프로젝트 회의록.txt => "전체 파일", "프로젝트"
